Análisis del Problema

1. Identificación del Problema

Las pequeñas y medianas empresas (PYMEs) enfrentan serias dificultades en la gestión de sus recursos humanos debido a la ausencia de herramientas digitales especializadas. La mayoría administra información del personal mediante hojas de cálculo, registros en papel o correos electrónicos, lo que genera desorganización, pérdida de datos y procesos lentos e ineficientes.

Problema central identificado:
Las PYMEs no cuentan con un sistema centralizado que les permita gestionar de manera eficiente la información de sus empleados, el control de asistencia, las solicitudes de vacaciones y la documentación del personal.

2. Árbol de Problemas

Falta de control automatizado de asistencia y puntualidad
Procesos manuales para solicitar y aprobar vacaciones
Ausencia de un sistema de roles y permisos para el personal
No existe un repositorio centralizado de documentos del personal

3. Selección del Problema

Tras analizar la situación de las PYMEs en el entorno local, se seleccionó este problema porque:

  • Afecta a la mayoría de empresas pequeñas que no pueden costear sistemas ERP empresariales costosos.
  • Los procesos manuales generan errores frecuentes, pérdida de tiempo y conflictos internos.
  • Existe una brecha tecnológica real entre grandes empresas (que sí tienen sistemas) y las PYMEs.
  • Una solución web accesible puede impactar positivamente en decenas o cientos de empresas.

Planteamiento de la Solución

4. Objetivo General

Desarrollar un sistema web de gestión de recursos humanos (HR Cloud Manager) que permita a las PYMEs administrar de forma digital, centralizada y segura la información de sus empleados, el control de asistencia, las solicitudes de vacaciones y la documentación del personal.

5. Objetivos Específicos

  1. Implementar un módulo de gestión de empleados con registro completo de datos laborales y personales.
  2. Desarrollar un sistema de control de asistencia con registro de entrada/salida y cálculo automático de tardanzas.
  3. Crear un flujo de solicitud y aprobación de vacaciones con estados (pendiente, aprobado, rechazado).
  4. Implementar un repositorio de documentos del personal con control de vencimientos.
  5. Diseñar un sistema de roles y permisos (admin, gerente, RRHH, empleado) con acceso diferenciado.
  6. Proveer un dashboard con indicadores estadísticos en tiempo real para la toma de decisiones.

7. Justificación

El sistema se justifica porque resuelve un problema real y frecuente en el contexto de las PYMEs: la falta de digitalización en la gestión del capital humano. Al implementar HR Cloud Manager, las empresas podrán:

  • Reducir el tiempo destinado a tareas administrativas manuales.
  • Eliminar errores en el registro de asistencia y cálculo de tardanzas.
  • Garantizar transparencia en los procesos de solicitud y aprobación de vacaciones.
  • Mantener un historial digitalizado y seguro de cada empleado.
  • Tomar decisiones basadas en datos actualizados y accesibles en tiempo real.
