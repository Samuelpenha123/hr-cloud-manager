🔗 Relaciones entre entidades

users → employees 1 : N
Un usuario (admin) gestiona muchos empleados.
employees.user_id = users.id
users → users 1 : N
Un admin puede crear múltiples sub-usuarios.
users.created_by = users.id
users → employees 1 : 1
Un usuario empleado está vinculado a su registro HR.
users.employee_id = employees.id
employees → attendances 1 : N
Un empleado tiene muchos registros de asistencia.
attendances.employee_id = employees.id
employees → vacation_requests 1 : N
Un empleado puede tener muchas solicitudes de vacaciones.
vacation_requests.employee_id = employees.id
employees → documents 1 : N
Un empleado puede tener muchos documentos adjuntos.
documents.employee_id = employees.id
users → vacation_requests 1 : N
Un usuario (admin/hr) aprueba/rechaza solicitudes.
vacation_requests.approved_by = users.id
users → notifications 1 : N
Un usuario recibe muchas notificaciones del sistema.
notifications.user_id = users.id
